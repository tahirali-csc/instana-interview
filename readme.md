# **Interview Exercise**

#### **Pre-requisites**
1. JDK 11
2. Maven >=3.6.3


#### **Compile & Run Tests**
This runs tests and produces interview-app.jar at the base directory.
```
./build.sh
```

### **Run the app**
The ```input.csv``` is located at the base directory.
```
java -jar interview-app.jar input.csv out.txt
```
